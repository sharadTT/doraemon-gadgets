The user must enter the booth, take the phone off, then say "What if ____?" , then set the phone down. Once this is done, it technically warps the user to an alternate dimension of their current world, while original world and its timeline continue on as usual. Traveling future after using the gadget will result in the user arriving at the same parallel world and not the original world, but traveling back to the past will result in the user arriving at the original world prior to the usage of the gadget.
